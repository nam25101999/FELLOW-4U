# Flutter & Dart Skill Tree

Lộ trình kỹ năng từ Junior đến Senior Developer.

## 🟢 Level 1: Junior (Nền tảng)
- [ ] **Dart Basics:** Variable types, Functions, Collections (List, Map, Set).
- [ ] **Asynchronous:** `Future`, `async/await`, cơ bản về `Stream`.
- [ ] **Widgets:** Nắm vững `Stateless` vs `Stateful`, common widgets (Row, Column, Stack, Container).
- [ ] **Layout:** Hiểu về `Constraints` (BoxConstraints) - "Constraints go down. Sizes go up. Parent sets position."
- [ ] **Navigation:** Sử dụng `Navigator` cơ bản.
- [ ] **API:** Biết cách dùng `http` package để fetch dữ liệu JSON.

## 🟡 Level 2: Intermediate (Trung cấp)
- [ ] **State Management:** Thành thạo ít nhất một thư viện (Riverpod, BLoC, hoặc Provider).
- [ ] **Clean Architecture:** Hiểu cách tách lớp (Data, Domain, Presentation).
- [ ] **Advance Networking:** Sử dụng `Dio`, Interceptors, và xử lý lỗi tập trung.
- [ ] **Local Storage:** Sử dụng `Hive`, `Isar`, hoặc `Sqflite`.
- [ ] **Animations:** Sử dụng `AnimationController`, `Tweens`, `Implicit Animations`.
- [ ] **Dependency Injection:** Sử dụng `get_it` hoặc cơ chế DI của Riverpod.
- [ ] **Testing:** Biết viết `Unit Test` cho Logic.

## 🔴 Level 3: Senior (Cao cấp)
- [ ] **Design Patterns:** Áp dụng SOLID, Factory, Singleton, Observer vào Flutter.
- [ ] **Performance:** Debug bằng DevTools, xử lý `Memory leaks`, `Jank`.
- [ ] **Native Bridge:** Viết `MethodChannel` (Android/iOS).
- [ ] **CI/CD:** Setup pipeline tự động (GitHub Actions, Fastlane).
- [ ] **Complex UI:** `CustomPaint`, `CustomRenderObject`.
- [ ] **Security:** SSL Pinning, Obfuscation, Secure Storage.
- [ ] **Micro-Frontends:** Chia nhỏ app thành các Packages/Modules độc lập.
